#### Search 

###### 基本规则
```angular2html
1. 默认搜索的是master分支：搜索语句不能有特殊字符：. , : ; / \ ` ’ ” = * ! ? # $ & + ^ | ~ < > ( ) { } [ ]
2. 检索关键字：(例如：android)
    - 按照文件搜素：android in:file
    - 按照路径搜素：android in:path
    - 按照语言搜素：android language:java
    - 按照文件大小：android size:>100
    - 按照后缀搜素：android extention:css
    - 按照是否被fork：android fork:true
    - 按照地域检索：android location:guangzhou
    
```
